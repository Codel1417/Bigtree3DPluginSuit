# Bigtree3DPlugin
Store gcode to SD or disk with Bigtree3D format
# 
## How to use ?
Copy this folder to the Cura plugins directory
* Find your Cura installation directory. For example, the default installation path is **C:\Program Files\Ultimaker Cura 4.6\plugins**
* Download this folder, unzip and copy this folder to the plugins folder of Cura
  ![image](https://user-images.githubusercontent.com/38851044/87514486-e52fa180-c6ac-11ea-89e1-872d3da7e77c.png)

* Open the STL model in Cura and click `Slice`
* Click the small up arrow on the right and select `Save as Bigtree3D format`
  ![image](https://user-images.githubusercontent.com/38851044/87514825-6dae4200-c6ad-11ea-9526-585325a38b33.png)
* Update touch screen firmware after this [Pull request](https://github.com/bigtreetech/BIGTREETECH-TouchScreenFirmware/pull/844), The first release to include this feature is Vx.x.27
* The actual effect is shown in the figure
  ![image](https://user-images.githubusercontent.com/38851044/87516630-47d66c80-c6b0-11ea-9831-aa67aa531a3d.png)
  ![image](https://user-images.githubusercontent.com/38851044/87516942-c3381e00-c6b0-11ea-9620-3397978b99fb.png)

## Which version of hardware supports this feature?
 * All versions of touch screen hardware released by BIGTREETECH support this feature as long as the firmware is updated, including TFT24/TFT28/TFT35/TFT43/TFT50/TFT70...
